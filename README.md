Zen Automotive Profile | Official Portfolio

「ぜん」のパーソナリティを高級車のダッシュボード風に表現した、ハイスクール・エディションの自己紹介サイトです。

🏎️ デザインコンセプト

「のんびり系」という穏やかな性格と、車のような「かっこよさ・メカニカルな美しさ」を融合させたデザインです。
高級車のコックピットをイメージしたダークモードを基調とし、カーボン調のテクスチャやネオンの光、動的な計器類（メーター）を取り入れています。

主な特徴

HUD（ヘッドアップディスプレイ）風ヘッダー: リアルタイムクロックとシステムステータスを表示。

動的なドライブモード表示: 「のんびり」とした雰囲気をメーター形式で視覚化。

レスポンシブ設計: スマートフォン、タブレット、PCのすべてのデバイスで最適化されたレイアウト。

物語性のある長文構成: 単なる箇条書きではなく、高校生活や趣味を車のスペックに見立てたストーリー形式で紹介。

🛠️ 技術スタック

HTML5: セマンティックな構造。

Tailwind CSS: 高度なスタイリングとレスポンシブ・グリッド・システム。

JavaScript: リアルタイムの時刻更新、スムーススクロール、および動的なUI演出。

Google Fonts: 「Exo 2」（メカニカルな欧文フォント）と「Noto Sans JP」を採用。

📂 セクション構成

Profile (Driver ID): 基本プロフィールと現在のドライブモード。

Specs (Performance Specs): 趣味（アニメ・読書・映画）と得意なこと（走ること）、エネルギー源（韓国料理）の紹介。

Mission (Roadmap): これから挑戦したいこと（料理）と現在のスキル習得状況のインジケーター。

Log Message: 閲覧者への感謝のメッセージ。

🚀 プレビュー方法

index.html をブラウザで開くだけで閲覧可能です。

Canvasの「Preview」ボタンからも直接動作を確認できます。
